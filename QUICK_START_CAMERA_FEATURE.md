### Camera Not Working
```
Problem: "Camera access denied"
Solution:
1. Check browser permissions
2. Go to Settings â†’ Privacy â†’ Camera
3. Allow EduBridge access
4. Refresh browser (F5)
5. Click "Enable Camera" button
```

### Text Not Visible
```
Problem: "Content too small or unclear"
Solution:
1. Use browser zoom (Ctrl + Plus)
2. Adjust window size
3. Click "Fullscreen" if available
4. Try different background (white/gray)
```

### Emotion Detection Not Working
```
Problem: "No emotion stats showing"
Solution:
1. Ensure face is in camera frame
2. Check lighting (not dark)
3. Look toward screen naturally
4. Wait 5-10 seconds for calibration
5. Refresh and try again
```

### Game Won't Start
```
Problem: "Can't open games"
Solution:
1. Check "Take a Break" button visible
2. Ensure you're on 5+ page documents
3. Update browser to latest version
4. Clear browser cache
5. Try different browser
```

### Can't Resume Reading
```
Problem: "Content reloads to top"
Solution:
1. Game modal should auto-close
2. Click "Continue Reading" button
3. Use page slider to navigate
4. Try again from "Take a Break"
```
